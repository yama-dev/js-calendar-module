## Using

### NPM Usage

``` bash
# install npm.
npm install --save js-calendar-module
```

``` javascript
// import.
import CALENDAR_MODULE from 'js-calendar-module';
```

### Basic Standalone Usage

``` html
<script src="./js-calendar-module.js"></script>
<script>
  let CM =  new CALENDAR_MODULE({
  });
</script>
```
